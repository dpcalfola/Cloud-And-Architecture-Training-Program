1. sudo
    - 관리자 모드로 실행하겠다는 키워드
    - 명령에어 가장 앞에 위치시켜서 명령어를 관리자 권한으로 실행
    - 설정이나 설치, 제거 등의 대부분의 명령어가 관리자 권한을 필요로 합니다



2. apt & apt-get
    - (데비안 계열)리눅스의 패키지 매니저 명렁어
    - 프로그램을 설치하겠다는 뜻


    * apt-get install [패키지명]
        # apt-get install openssh-server
           -> openssh-server 설치한다는 뜻
           -> 제일 뒤에 -y 를 붙이면 모든 질문에 자동 Yes를 입력


3. pwd
    - 현재 위치를 알려주는 명령어


4. ls
    - 현재 디렉토리의 파일과 디렉터리 목록을 출력
    - ls -a : 숨김 파일까지 출력


5. cd
    - 디렉토리를 이동하는 명령어
        - cd .. : 상위 디렉토리로 이동
        - cd ~ : 홈 디렉토리로 이동
        - cd / : 루트 디렉토리로 이동
        - cd [디렉토리명] : 해당 디렉토리 이동 (상대경로 이동)
        - cd /[디렉토리명]/[디렉토리명]... : 절대경로 디렉토리 이동


6.  vi [파일명]
    - 파일의 내용을 편집
        - (esc -> i) 입력모드
        - (esc -> :wq) 저장 후 종료
        - (esc -> :q!) 저장하지 않고 종료
    - 일치하는 파일 명이 없을경우 파일을 생성하고 편집기 실행


7. cat [파일명]
    - 파일의 내용을 출력


8. touch [파일명]
    - 파일을 생성하는 명령어
    - 생성만하고 편집은 하지 않음, 비어있는 파일만 생성


9. cp
    - 파일 복사(copy)
    - cp [원본파일] [복사파일]


10. rm
    - 파일 또는 디렉터리 삭제(remove)
    - rm [파일명]
    - rm -r [디렉토리명] : 디렉토리 삭제
