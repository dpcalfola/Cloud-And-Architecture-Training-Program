< DB 생성 >

MySQL
    - innoDB version



    - 고가용성 여부: 설정시 2개 서버 생성
    - VPC 설정
    - 멀티존 구성 가능
    - 두개의 서브넷 설정 가능 (서로 다른 존에 서버 생성)
    - 암호화 여부
    - 스토리지 타입: SSD, HDD
    - 10GB시작 -> 10GB씩 증가 -> 최대 6TB
    - Private Subdomain 설정 가능

    - ACG 자동 설정됨

    - DB config 설정 (NCP 에서 제공하는 설정값 - 수정 불가)
    - DB log -> CLA

    - 백업 파일 보관기능
    - 백업 시간 설정 가능 (15분 단위)

< Cloud Outbound mailer >

1. Manage Template
    - Template Unique ID
    - name
    - 송신자 정보
    - 제목
        - 개인화된 제목 설정 가능 (치환태그)
        - ${name}님을 위한 ${service} 서비스 안내
    - 내용
        - 내용에도 치환태그 사용 가능
    - 치환태그 value
        -> file upload 로 가능
        -> 입력 양식 제공
        -> Template 에 사용된 치환태그를 반영하여 생성된 엑셀 형식의 양식이 제공됨
